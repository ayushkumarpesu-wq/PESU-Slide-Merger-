🔧 TECHNICAL DETAILS:
═══════════════════════════════════════════════════════════════════════════

Backend Architecture:
   Browser ↔ Flask API ↔ PESU Academy
   
   - REST API endpoints
   - Background thread processing
   - Thread-safe state management
   - CORS enabled for security

Frontend Architecture:
   HTML ↔ CSS ↔ JavaScript ↔ Fetch API
   
   - Pure vanilla JavaScript (no frameworks)
   - CSS Grid & Flexbox layout
   - Fetch API for AJAX
   - Real-time polling (500ms)

Data Flow:
   1. User selects options
   2. JavaScript validates input
   3. Fetch sends to Flask API
   4. Flask spawns download thread
   5. Main thread returns response
   6. JS polls status every 500ms
   7. UI updates in real-time
   8. Downloads complete

🆘 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════

Port 5000 already in use?
   Edit web_app.py, change:
   app.run(debug=True, host='0.0.0.0', port=5001)

Flask not starting?
   ✓ Check Python: python --version
   ✓ Check dependencies: pip list | grep Flask
   ✓ Try: pip install Flask Flask-CORS

Can't access localhost:5000?
   ✓ Ensure Flask is running (no errors in terminal)
   ✓ Try: http://127.0.0.1:5000
   ✓ Check firewall settings

Login fails?
   ✓ Check .env file (PESU_EMAIL and PESU_PASSWORD)
   ✓ Verify credentials work on PESU website directly
   ✓ Check internet connection

Download stuck?
   ✓ Check internet connection
   ✓ Refresh browser (F5)
   ✓ Restart Flask server

No progress updates?
   ✓ F12 → Console (check for errors)
   ✓ Refresh page (F5)
   ✓ Check Flask logs in terminal
